# Reproduction Plan

## Goal
Reproduce SentiMM as a runnable framework with modular five-stage agents and explicit ablation switches.

## Steps
1. Build SentiMMD-like dataset interface.
2. Implement five agents with local open-source components.
3. Build train/eval pipeline and metric reporting.
4. Add ablation runner for required five settings.
5. Add tests + smoke test.
6. Document assumptions and boundaries.
